# disord-python-bot-template
- Il s’agit d’un modèle de bot discord.py destiné à être utilisé avec l’egg discord.py inclus pour le panel Pterodactyl.

# Installation
1. Importez d’abord le fichier JSON [egg-discord-python-bot.json](egg-discord-python-bot.json) dans le panel Pterodactyl en tant que nouvel egg.  
   Pour plus d’informations, veuillez consulter [ce site](https://pterodactyl.io/community/config/eggs/creating_a_custom_egg.html).
2. Redémarrez le daemon après avoir ajouté le nouvel egg.
3. Ensuite, créez un nouveau serveur en utilisant cet egg ; vous devrez renseigner le token du bot ainsi que le préfixe du bot.
4. Allez sur le panel et démarrez le bot.
